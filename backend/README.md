# Ayipoee Coffee Shop Backend

REST API endpoints:
- GET /api/health
- GET /api/products
- POST /api/products
- POST /api/orders
- GET /api/orders

## Run
1. cd backend
2. npm install
3. npm run dev  # uses nodemon

Backend assumes MongoDB is running locally at mongodb://127.0.0.1:27017/ayipoee_coffee_shop
