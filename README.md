# Ayipoee Coffee Shop

Full-stack coffee ordering application with Node.js backend and MongoDB.

## Features
- Coffee menu display
- Order placement with product selection
- MongoDB cloud database (Atlas)
- Real-time order management

## Running Locally
```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npx serve .
```

Open http://localhost:3000

## Technology
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB Atlas
